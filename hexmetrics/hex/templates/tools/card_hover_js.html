HEX Card Hover Javascript

What is it?

The Card Hover javascript is a tool for website owners, so that they can include card tooltips on mouseover, like this.

How Do I Use It?

Getting the mouseovers to work is fairly straightforward, first - add the following snippet between your <head> and </head> tags: <script src="http://hexmetrics.ni.tl/js/hex-card-hover.js"></script>

Next, you link to a HEX Wiki card like http://hextcg.gamepedia.com/Phoenix_Guard_Aeronaut and use the class 'hex-card', like so: <a class="hex-card" href="http://hextcg.gamepedia.com/Phoenix_Guard_Aeronaut">Phoenix Guard Aeronaut</a>

Now, when someone mouses over your link, you get Phoenix Guard Aeronaut.

Something doesn't work!

The tool is currently under active development, if you need further help, please leave a comment below. Additionally, if you can help out with any part of the process, please let me know - there are a couple of glaring omissions at the moment, such as card colours and cart art.
